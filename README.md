
# End-to-End CI/CD Pipeline on AWS

## Project Summary

Built an automated CI/CD pipeline to deploy a containerized Flask application on AWS EC2 using Jenkins and Docker.

The pipeline automatically builds and deploys the application whenever code is pushed to GitHub.

---

## Tech Stack

- AWS EC2
- Jenkins
- Docker
- Python (Flask)
- Git & GitHub
- Linux

---

## What This Project Does

✔ Developer pushes code to GitHub  
✔ Jenkins pulls latest code  
✔ Docker image is built automatically  
✔ Old container is stopped  
✔ New container is deployed on EC2  
✔ Application goes live instantly  

---

## Architecture Flow

GitHub → Jenkins → Docker → AWS EC2 → Live Application

---

## Deployment

Application runs on:

http://<EC2-PUBLIC-IP>:5000

---

## Key Skills Demonstrated

- Continuous Integration & Deployment  
- Containerization using Docker  
- Cloud Deployment on AWS  
- Jenkins Pipeline Automation

---

## Outcome

Reduced manual deployment effort and achieved automated, repeatable production-ready deployments using CI/CD principles.
